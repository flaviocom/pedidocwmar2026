# 🛒 Manual de Uso Básico: Sistema de Pedidos de Vendas

Este é um guia rápido e simplificado com as principais funcionalidades para facilitar a emissão de pedidos no site:

## 1. Iniciando um Novo Pedido
- **Limpar Dados:** Para iniciar rapidamente um atendimento, clique no primeiro botão de **"Limpar"** (ícone de lixeira no topo). Isso zerará os dados anteriores do formulário e puxará a data atual de forma automática.

## 2. Preenchendo os Campos Obrigatórios
- **Atendimento por:** É de extrema importância preencher o nome de quem está fazendo o atendimento. **Atenção:** se este campo ficar vazio, o sistema não deixará você salvar o pedido ao final.
- **Dados do Cliente:** Preencha o nome completo do cliente correspondente.

## 3. Seleção de Produtos e Valores
- **Quantidade dos itens:** Vá à linha de cada produto correspondente e coloque a quantidade. A partir desse momento, todo o cálculo do pedido — como valor total, lucro, etc. — será gerado de maneira automática.
- **Desconto Geral:** Caso queira aplicar uma margem de desconto para toda a venda, basta preencher a caixa de desconto principal.

## 4. Desconto Especial por Produto (Atenção)
- Se você possui uma oferta ou deseja dar um desconto diferenciado apenas para um item específico, altere a porcentagem de desconto na **própria linha do produto**.
- **Regra de Atualização:** Sempre que esse desconto for alterado diretamente na linha de um produto, é necessário **clicar na lixeirinha no alto do pedido** para que o sistema acate este novo desconto especial e atualize os cálculos.

## 5. Venda para Consumidora Final (Sem Desconto)
- O sistema é muito flexível e também pode ser perfeitamente usado para calcular pedidos para o varejo comum/cliente final (**consumidoras**) e não somente revendedoras.
- **Como fazer:** Basta **não inserir desconto nenhum**. Dessa forma, não aparecerá nenhuma menção a descontos na interface e o comprovante do pedido ficará com um aspecto muito limpo ("bonitinho"), ótimo para ser enviado diretamente à cliente pelo **WhatsApp**!

## 6. Salvando e Historificando
- **Salvar Pedido:** Ao finalizar de montar a venda, certifique-se de clicar no botão **"Salvar Pedido"**. Isso é o que armazena os dados da venda de forma segura no seu **Histórico**.
